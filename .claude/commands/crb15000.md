variants show), not a manual-stated number — still checked 2026-07-24, still not resolvable to
an exact figure from this source.

## Mounting

| Position | Installation angle |
|----------|--------------------|
| Floor | 0° |
| Wall | Any angle |
| Suspended (inverted) | 180° |

Mounting angle **must** be set in system parameters, otherwise performance and lifetime are affected.

Foundation requirements: flatness 0.1/500 mm, minimum resonance frequency 22 Hz, yield strength ≥150 MPa.

Attachment hardware (included in delivery):
- 4× M10×35 screws, quality 8.8
- Washer 23/10.5/2.5 mm steel
- 2× guide pins DIN6325, Ø6×24 mm
- Tightening torque: 32 Nm ±10%

**Do not change robot position before securing it to the foundation.**

## Safety

- Safety category: **3**, performance level **d** (EN ISO 13849-1 / EN ISO 10218-1:2011)
- All safety functionality lives in the joint units — only replaceable as complete units
- Cyclic Brake Check must run every **8–48 hours** (see manual p.161)
- Reaction times: stopping functions 4 ms, position/torque monitoring 6 ms, brake closing <110 ms

## Coordinate System (used in RWS and RAPID)

All pose data from RWS `robtarget` is in the **Base** coordinate frame:
- Origin at centre of robot base mounting flange
- Z-axis points up (for floor-mounted robot)
- X-axis points toward robot front

Pose format from `gofa-pose` node:
```
{ x, y, z (mm), q1, q2, q3, q4 (quaternion), cf1, cf4, cf6, cfx (config flags) }
```

GOTO command format sent over TCP (11 semicolon-separated values), prefixed with a move type — `J` for MoveJ (joint-interpolated, more predictable — used whenever move type is unset/invalid) or `L` for MoveL (straight-line TCP path, can hit singularities/joint limits along the line):
```
GOTOJx;y;z;q1;q2;q3;q4;cf1;cf4;cf6;cfx
GOTOLx;y;z;q1;q2;q3;q4;cf1;cf4;cf6;cfx
```
Bare `GOTOx;y;z;...` (no `J`/`L`) is still accepted as an alias for `GOTOJ`. Values are rounded to 1 decimal place (xyz) and 4 decimal places (quaternion) to stay within RAPID's 80-character string limit.

## Shipping / Home Positions

Default shipping position for CRB 15000-5/0.95: A1=0°, A2=0°, A3=+85°, A4=0°, A5=0°, A6=0°  
For CRB 15000-10/1.52: A1=0°, A2=–15°, A3=–225°, A4=0°, A5=–30°, A6=0°  
For CRB 15000-12/1.27: A1=0°, A2=–10°, A3=+85°, A4=0°, A5=+15°, A6=0°

Project home position: saved to `HOME:/Programs/gofa_home.cfg` on controller via `SETHOME` command.
